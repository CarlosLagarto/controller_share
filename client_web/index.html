Sobre a Aplicação - Sistema de Domótica/IoT

Self-regulated IoT AI automation system.

Sistema de automação inteligente autoregulado.


© Lagarto - Computer Science & Systems Engineering


Porquê: Para dar asas ao meu espirito "nerd", e atualizar conhecimento de HTML5, CSS3, SVG, RUST, Python, Javascript ECMA2015, WebApps, Apache, MySql, SqLite, Linux, ...


Sistema de controlo inteligente para a rega, aquecimento central, controlo e integração de sensores de ambiente e metereologia, vigilância, deteção de intrusão, com cenários de automação.


Como funciona: Sistema distribuído, assíncrono, com near real-time controller desenvolvido em RUST, num mini-pc intel/ubuntu. O controller faz bridge/integra com dispositivos externos, nomeadamente:

- Relés com ESP8246 para interface de controlo com as electroválvulas da rega, aquecimento central, estores, etc.

- Mosquitto MQTT Broker near real-time clients integration.

- Apache Web para servir a aplicação cliente WebApp PWA

- Green IT - para manter o consumo de energia abaixo de 0.5% de utilização de CPU.

- Fail Safe - Redundância dos mecanismos de controlo

- Tempest - Estação local com informação a cada minuto.


Quando funciona: Elevada disponibilidade - 24x7x365. Estatísticas a disponibilizar quando as houver.

Sistema não desenhado para esta orientação/dispositivo.

Utilize um dispositivo com ecran superior a 320px ou se estiver em modo "landscape" utilize o telefone em modo "portrait".
